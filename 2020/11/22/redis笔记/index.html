什么是nosql

nosql: not only sql 泛指非关系型数据库

随着web2.0互联网的诞生，传统的关系型数据库很难对付2.0时代。

nosql特点

  1. 方便扩展（数据之间没有关系，很好扩展）
  2. 大数据高性能
  3. 数据类型多样性（不需要事先设计数据库，随取随用）
